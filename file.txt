❯ what is the issue above                                                       

● Two issues:

  1. python3 permission denied — python3 on your system points to the Windows
  Store stub (WindowsApps/python3), which is blocked in the bash shell. This is
  a common Windows issue.
  2. python 3 typo — The space made it treat 3 as a file argument, not part of
  the command.

  Fix: Use python instead of python3:

  python -m venv venv

  Your Python 3.13 is installed at C:\Program Files\Python313\python.exe and is
  accessible via python.

      └── js/main.js          # Frontend JavaScript

  How it fits together:

  - app.py is the entry point. It creates the Flask app and defines routes like
  /, /register, /login. Several routes (logout, profile, add/edit/delete

  - base.html is the shared layout that all other templates extend — it includes
   the navbar, footer, fonts (DM Serif + DM Sans from Google Fonts), and loads
  the CSS/JS.
  - database/db.py is a placeholder — the SQLite connection, table creation, and
   seed data are meant to be written there as part of the project steps.
  - The CSS uses CSS custom properties (variables) for a consistent design
  system with a warm paper-toned theme.
